# Ciel Website

Landing page for [Ciel](https://github.com/shubham-pathak1/ciel), a high-performance download manager for Windows.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Deployment

This site is designed to be deployed on [Vercel](https://vercel.com).
